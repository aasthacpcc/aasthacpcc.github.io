# Week 3 – Themes, Menus, Widgets & Customization

## Objective
- Customize the site’s theme, navigation, and layout for a functional and responsive website.

---

## Topics Covered
- Custom menus and navigation
- Static homepage setup
- Theme customization (fonts, colors, header image)
- Add widgets to sidebar/footer
- Responsive design basics

---

## Tasks / Exercises
1. Customize your **WordPress.com theme** (colors, fonts, header image).
2. Set a **static homepage**.
3. Create a **main menu** linking key pages.
4. Add **widgets** to sidebar and footer.
5. Check site **responsiveness** on smaller screen sizes.

---

## Project Ideas
- Portfolio with gallery
- Business site with product or service pages
- Blog with multiple categories
- Event website with schedule and posts

## Outcome
- Functional WordPress site with static homepage.
- Navigable menus and widgets added.
- Theme customized and responsive.
