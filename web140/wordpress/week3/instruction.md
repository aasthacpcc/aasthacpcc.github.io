# Week 3 – Themes, Plugins & Customization

## **Objective**

Learn to customize a live WordPress.com site using themes and plugins, and practice managing content access through user roles.

---

## **Topics**

* Exploring free WordPress.com themes
* Customizing themes using the WordPress Customizer
* Installing and configuring essential free plugins:

  * Contact Form 7 (forms)
  * Yoast SEO (SEO)
  * Jetpack (stats, social sharing)
* Adding widgets to sidebar/footer
* Understanding and reviewing user roles for plugin-based content access

---

## **Tasks / Exercises**

1. Select and install a free theme on your WordPress.com site.
2. Customize the theme using the WordPress Customizer:

   * Update site title, tagline, and colors
   * Adjust header/footer layout
3. Install the following plugins:

   * Contact Form 7: create a Contact page form
   * Yoast SEO: configure for your homepage and posts
   * Jetpack: explore and enable at least one feature (e.g., social sharing or stats)
4. Add widgets to the sidebar or footer (e.g., recent posts, search bar, or categories).
5. Review your users’ roles and adjust permissions if needed for plugin content.

---

## **Website Ideas**

* **Personal Portfolio** – Add a contact form and social links with Jetpack.
* **Blog** – Configure SEO for posts and add sidebar widgets.
* **Small Business Demo** – Include a contact form, newsletter form, and widget for promotions.
* **Club or Event Page** – Add a registration/contact form and sidebar events list.

---

## **Submission Requirements**

* Submit the **WordPress.com site link** showing all customizations, plugins, and theme applied.

---

## **Outcome**

* Students will have a fully customized live WordPress.com site with:

  * Selected and personalized theme
  * Configured plugins
  * Widgets added
  * User roles verified for plugin content access
* Students will gain confidence in **enhancing site functionality and appearance**.
