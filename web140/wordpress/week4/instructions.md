# Week 4 – Advanced Features & Publishing

## **Objective**

Implement advanced site features on your WordPress.com site, including homepage layout, forms, SEO optimization, image performance, security, and user role management.

---

## **Topics**

* Creating a functional homepage layout
* Internal linking and navigation between pages/posts
* Installing additional plugins:

  * WPForms Lite (newsletter/contact forms)
  * Smush (image optimization)
* Basic SEO best practices
* Security review: strong passwords, 2FA, plugin updates
* User role adjustments and permission review

---

## **Tasks / Exercises**

1. Design a functional homepage layout incorporating text, images, and menus.
2. Add **internal links** between pages and posts for smooth navigation.
3. Install and configure additional plugins:

   * WPForms Lite: create a newsletter signup or contact form
   * Smush: optimize images for faster load times
4. Review and optimize SEO using Yoast SEO for homepage and posts.
5. Conduct a **security check**:

   * Verify strong password and 2FA enabled
   * Ensure all plugins/themes are updated
6. Review user roles and adjust permissions for contributors/editors as needed.

---

## **Website Ideas**

* **Personal Portfolio** – Add a homepage with featured projects and newsletter signup.
* **Blog** – Optimize images, set up newsletter, and enhance navigation.
* **Small Business Demo** – Homepage with contact form, product images, and internal links.
* **Club or Event Page** – Event homepage with registration form, images optimized, and sidebar info.

---

## **Submission Requirements**

* Submit the **WordPress.com site link** showing all advanced features, homepage layout, forms, SEO optimization, security settings, and user roles applied.

---

## **Outcome**

* Students will have a **fully functional live WordPress.com website** with:

  * Homepage layout designed
  * Posts and pages interconnected
  * Forms for user interaction
  * Optimized images and SEO
  * Verified security settings
  * Properly managed user roles
* Students will gain experience managing a complete WordPress site ready for public use.
