## Objective
- Learn WordPress basics and set up a local site using LocalWP.
- Decide on a project topic and prepare a short project proposal.

---

## Topics Covered
- WordPress.com vs WordPress.org
- Introduction to LocalWP
- WordPress Dashboard overview
- Pages vs Posts
- Creating content (text, images, links)
- Turn on comments

---

## Tasks / Exercises
1. Install LocalWP and create a local site (`MyLocalSite`).
2. Explore the dashboard:
   - Pages
   - Posts
   - Media
   - Menus
   - Themes
3. Create **one page** called `Home`.
4. Create **one post** with text and an image.
5. Enable **comments** on posts.
6. Write a **short project proposal**:
   - Project title
   - Tagline
   - Planned pages
   - Categories or topic
7. Optional: Explore different themes and menus to understand structure.

---

## Project Ideas
- Personal blog
- Small business demo website
- Club or event website
- Online portfolio (for non-personal topics, e.g., art, writing, or coding projects)
