## Objective
- Set up a live WordPress.com site and create pages, posts, and media-rich content.

---

## Topics Covered
- Gutenberg Block Editor (Text, Media, Design Blocks)
- Adding images, video, PDF
- Featured images
- Categories & Tags
- Turn on comments

---

## Tasks / Exercises
1. Create a **free WordPress.com account**.
2. Set up a new **live site** (e.g., `MyLiveSite`).
3. Explore the dashboard:
   - Pages
   - Posts
   - Media library
   - Menu creation
   - Themes
4. Create **two pages**:
   - Home
   - About
5. Create **two posts** with media (image/video/PDF) and set a **featured image**.
6. Enable **comments** on posts.
7. Add **categories and tags** to posts.
