# GitHub Setup Assignment

## **Objective**
Learn how to create a GitHub account, repository, and folder structure so that you are ready to upload your web project files later.

---

## **Objective**
Create your GitHub account, set up your `username.github.io` repository, create an `index.html` file, link it to your `web140` folder/repository, and publish your first live website using GitHub Pages.

---

## **Part 1: Create a GitHub Account**
1. Go to **https://github.com**
2. Sign up with a valid email address  
3. Verify your email  
4. Log in to your new GitHub account  

---

## **Part 2: Create Your Main Repository**
This repository will host your **GitHub Pages website**.

1. Click **New repository**
2. Name the repository exactly: yourusername.github.io

*(replace “yourusername” with your actual GitHub username)*  
3. Add a description (optional)  
4. Select **Public**  
5. Check **Initialize this repository with a README**  
6. Click **Create Repository**

---

## **Part 3: Add Your `index.html` File**
This will be the homepage of your GitHub Pages website.

1. Inside your new repository, click **Add file → Create new file**
2. Name the file: index.html
3. Add content to the file (example):

```html
<h1>Welcome to My GitHub Site</h1>
<p>This site will link to my WEB140 projects.</p>
<p><a href="https://github.com/yourusername/web140">Visit my WEB140 repository</a></p>```
4. Click Commit changes

# Part 4: Create Your **web140** Repository or Folder

You may choose **either** option below:

## **Option A — Separate Repository (Recommended)**

1.  Create a new repository named:
    
    `web140`
    
2.  (Optional) Add a **README** file.
    
3.  This repository will contain **all WEB140 assignments**.
    

## **Option B — Folder Inside Your Main Repo**

1.  Go to your **`yourusername.github.io`** repository.
    
2.  Click **Add file → Create new file**.
    
3.  In the filename box, type:
    
    `web140/placeholder.txt`
    
4.  Commit the file.
    
5.  GitHub will automatically create the **web140** folder.
    

# Part 5: Enable GitHub Pages

1.  Go to your **`yourusername.github.io`** repository.
    
2.  Click **Settings → Pages**.
    
3.  Set:
    
    -   **Source:** Deploy from branch
        
    -   **Branch:** main
        
4.  Click **Save**.
    

Your live site will appear at:

`https://yourusername.github.io/`

# Important Note for Returning Students

If you previously took **WEB115, WEB215, or WEB250**, you may already have:

-   A GitHub account
    
-   A **username.github.io** repository
    
-   GitHub Pages already enabled
    

**If so, do NOT create a new one.**  
Simply update your existing **index.html** file to include the **WEB140 link**.
