# Image Editing & Showcase Assignment

**Objective:**  
Learn image editing using **Free Canva** and showcase your work online with before-and-after comparisons on a GitHub Pages website.

---

## Part 1: Image Editing

1. Choose **10 images** (personal photos or stock photos).  
2. For each image:  
   - Save the **original image**.  
   - Edit the image in **Free Canva** using at least one technique from the list below:
     - Crop & Resize  
     - Filters & Effects  
     - Text Overlay  
     - Background Change / Blur  
       > **Tip:** If you don’t have Canva Pro, use [remove.bg](https://www.remove.bg/) to remove backgrounds for free, then upload back to Canva.  
     - Stickers & Shapes  
     - Brightness, Contrast, Saturation  
     - Frames & Borders  
     - Transparency  
     - Collage / Multi-image Layout  
     - Creative Effects (Duotone, Glow)  
   - Save the **edited version**.  
   - Write a short description (1–3 lines) explaining what edits were applied.

---

## Part 2: Showcase Website

1. Create a GitHub repository: **`image-editing`**.  
2. Create an `index.html` file to display your images.  
3. For each image, show:  
   - Original vs Edited image  
   - Description of edits applied  
4. Push all original + edited images along with `index.html` to GitHub.  
5. Enable **GitHub Pages** and share the live URL.

---

## Deliverables

- GitHub repository containing:
  - `index.html`  
  - 10 pairs of original + edited images  
- Live GitHub Pages URL showcasing before-and-after comparisons

---

## Evaluation Criteria

- Creativity and skill in image editing  
- Clear comparison between original and edited images  
- Proper explanation of edits  
- Clean and functional portfolio layout
