# Assignment 2 – Figma High-Fidelity Design + Prototype

## Objective

Convert your wireframes into a high-fidelity UI design and create a clickable prototype using Figma.

## Requirements

### 1. Create a New Figma File or Duplicate Your Wireframes

* Name it: `YourName–Prototype`

### 2. Turn Your Wireframes Into Full Designs

For each screen:

* Add your color palette
* Add typography
* Add real or placeholder images
* Add buttons, icons, sections
* Use consistent spacing and alignment
* Optional: Use auto-layout for clean structure

### 3. Create at Least ONE Component

Examples:

* Button
* Card
* Navigation bar
* Footer

### 4. Build a Clickable Prototype

Use Prototype mode in Figma to:

* Link buttons to other screens
* Add simple transitions (e.g., “On Click → Navigate to”)
* Make the app/website flow realistic

**Required links:**

* Home → Screen 2
* Some reverse navigation (e.g., Back button)
* At least 3 total interactions

## Submission Requirements

* Public Figma link to the full project
* 3–5 fully designed screens
* Functioning prototype
* Short explanation (3–5 sentences) describing:

  * Your design choices
  * Your component
  * What is connected in your prototype

## Evaluation Criteria

* **Design Quality (40%)**

  * Layout, spacing, typography, color choices
* **Technical Skills (30%)**

  * Components used properly
  * Clean arrangement
  * Working prototype
* **Creativity (20%)**

  * Visual identity and theme
* **Explanation (10%)**

  * Clarity and completeness of reflection
